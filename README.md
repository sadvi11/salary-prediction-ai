# Salary Prediction AI

This is a beginner Machine Learning project that predicts salary based on years of experience.

## Technologies
Python  
scikit-learn  
Pandas  
Streamlit  

## Project Structure

salary-prediction-ai

dataset.csv  
train_model.py  
app.py  
requirements.txt  
README.md  

## How to Run

Install dependencies

pip install -r requirements.txt

Train the model

python train_model.py

Run the app

streamlit run app.py
